## What is it?
MiniJoyse is a line-based DSL according to ChatGPT, built in Python made by [perhapsjayimist](https://github.com/perhapsjayimist)
you can also check out the repo [here](https://github.com/perhapsjayimist/MiniJoyse/)
## How can I install it?
**There is no official package for package managers!**
What I mean by install is to make MiniJoyse work on the terminal.
I want YOU to place the project to `~/.local/share` and rename it to **MiniJoyse** so scripts can recognize it.
In the scripts folder you can put the run_nix.sh to `/usr/bin` or `/bin` and rename it to **minijoyse**
If you don't trust me then do not do it and run your .mjy scripts in here.
## How can I run it?
Run MiniJoyse using the run script inside the 'scripts' folder:
`python scripts/run tests/example.mjy/`
If you installed it you can type `minijoyse` on your terminal to run it too.
`minijoyse /path/to/example.mjy`
